# 淘宝用户行为深度分析与增长策略实战
## 一、 项目背景
### 基于2017年淘宝用户行为数据集，通过全流程的数据分析，挖掘用户行为模式，评估业务提出建议，并为运营策略提供数据支持。
## 二、 分析步骤
### 1.数据加载并清洗：处理时间戳，提取时间特征。
### 2.探索性分析(EDA)：宏观把握流量分布与用户活跃度。
### 3.提取核心业务指标：计算转化率、留存率，构建漏斗模型。
### 4.机器学习分层：利用 RFM 模型与 K-Means 聚类识别高价值用户。
## 三、 技术栈
### 数据处理: Pandas, NumPy
### 可视化: Matplotlib, Seaborn
### 机器学习: Scikit-learn (K-Means, PCA)
